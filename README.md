## 💡 About This Project
This website serves as my digital resume and showcases my featured data science and machine learning projects. It is designed with a modern, data-driven aesthetic, including custom animated background data flows and interactive data visualizations.

## 🛠️ Built With
This portfolio was built from scratch without heavy UI frameworks to ensure high performance and full control over the design:
*   **HTML5** - Semantic structure and content
*   **CSS3** - Custom styling, CSS Grid/Flexbox, glassmorphism effects, and smooth animations
*   **JavaScript (Vanilla)** - Interactive elements, scroll animations, modals, and dynamic components
*   **Feather Icons** - Minimalist and lightweight SVG icons

## 📁 Key Features
*   **Dynamic Data Background:** Animated SVGs and data flow representations mimicking a machine learning pipeline (Data Extraction -> Cleaning -> Analytics -> Insights).
*   **Preloader:** A visually engaging mock "data processing" preloader.
*   **Interactive Project Modals:** Dedicated modals displaying detailed information about each project (tech stack, key features, and links).
*   **Dark/Light Mode:** Full support for a toggleable theme that respects user preferences.
*   **Responsive Design:** Fully optimized for mobile, tablet, and desktop viewing.
*   **Integrated Resume Viewer:** Built-in PDF viewer for direct in-browser resume reading without forced downloads.

## 🏃‍♂️ How to Run Locally
